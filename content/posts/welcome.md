# 欢迎来到我的博客

这是我的第一篇博客文章！这个博客使用 **Next.js** 和 **Tailwind CSS** 构建，部署在 GitHub Pages 上。

## 关于这个博客

这个博客具有以下特点：

- 📝 使用 Markdown 编写文章
- 🚀 静态生成，加载速度快
- 🎨 简洁现代的设计
- 📱 响应式布局，支持移动设备

## 技术栈

- **Next.js 16** - React 框架
- **Tailwind CSS 4** - 样式框架
- **Bun** - JavaScript 运行时和包管理器
- **GitHub Pages** - 静态网站托管

## 开始写作

要添加新文章，只需在 `content/posts` 目录下创建新的 `.md` 文件，然后在文件头部添加 frontmatter：

```markdown
---
title: "文章标题"
date: "2026-03-25"
excerpt: "文章摘要"
tags: ["标签1", "标签2"]
---

文章内容...
```

感谢阅读！
